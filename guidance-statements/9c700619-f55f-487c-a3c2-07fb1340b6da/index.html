Cultural conventions, words, symbols

As a user with difficulty understanding content provided in unfamiliar or with unfamiliar conventions, I need content and interfaces that use standard conventions, words, and symbols for my culture, with which I am familiar. As a user with a congnitive disability, I must be able to change input and user interface components to use text or symbols with which I am familiar.
