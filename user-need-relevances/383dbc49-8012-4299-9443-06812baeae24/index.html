User Need Relevance: Voice Input

There are 2 guidance statements that support this user need relevance:

  • Speech variations - As a user with non-typical speech characteristics, such as from voice production difficulty or a Deaf accent, I need voice input systems to recognize my input as well as it does mainstream accents.
  • Voice Input Errors - As a user with a mobility disability who uses voice control, I need to minimize mistakes that could occur because of mistaken speech (e.g. operations are tied to a single letter shortcut)
