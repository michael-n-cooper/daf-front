Digital Accessibility Framework

This resource accesses a database of accessibility statements generated in the Digital Accessibility Framework project and presents it in a matrix of functional needs, user needs, and user need relevances. Individual categories of content are:

  • Categories
    • Functional need categorization
  • Matrix dimensions
    • Functional need
    • Intersection need
    • User need
    • User need relevance
  • Mapped statements
  • Term lists
    • Reference types
    • Tags
  • References
