Digital Accessibility Framework

This resource accesses a database of accessibility statements generated in the Digital Accessibility Framework project and presents it in a matrix of functional needs, user needs, and user need relevances.

Matrix

The matrix shows guidance statements and their relationship to the above dimensions. Links go to greater detail about the item and its relationships. This content can also be accessed by category:
