Digital Accessibility Framework

This resource accesses a database of accessibility statements generated in the Digital Accessibility Framework project and presents it in a matrix showing intersections of related aspects of the digital accessibility space. Refer to that site for an introduction to the structure of the information below.

Data in the framework

  • Accessibility statements - simple statements of an individual characteristic and needed environment affordance
    • Notes about the statement
    • References to research and guidelines
    • Tags for categorization
  • Framework dimensions
    • Individual abilities by group | Full list of individual abilities - Characteristics of the person using digital content
    • Object characteristics by group | Full list of object characteristics - Properties of the digital content being presented
    • Support needed - Type of accessibility requirement at the intersections of the above
  • Matrix of the entire data set

Feedback

Pages for accessibility statements and framework dimensions include a "Comment on this item" link which opens a form to submit your comment on that specific item. If you want to comment more generally, use one of the following forms:

  • General comment form
  • Accessibility statement suggestion form
